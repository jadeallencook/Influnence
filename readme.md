# Influencer V0.0.1

Set of scripts that help you hack your social presence and automate your goals.

## Installation

1. Launch the Chrome browser
1. Open ```chrome://extensions/```
1. Flip on "developer mode"
1. Choose "load unpacked"
1. Redirect to this repo
1. Choose ```extension/```
1. **Complete!**
